Language Set-Shifting Task (LS-ST) 


An English language based approach to the Wisconsin Card Sorting Test designed to target linguistic control cognition



##  Test procedure  ##
User must pick cards from hand that match target card
User must must go through 5 successful corrects to satisfy the rule shift streak requirement
Card in hand can be picked by either clicking with mouse (chosen will be highlighted) or by presssing number on KB
press "b" to show current rule + streak + timer (all this is hidden by default)
presss "d" to switch to dark mode, presss again for light
esc will close the program at any given time

##  The 3 rule dimesnsion  ##

1. Semantic category
- This category involves words with close contextual meaning / fall within a neat set when put togethor


2. Syntactic structure
- follow similar grammatical use

3. Phonological Features
- sound properties of words


##  Inventory Paramters  ##
**  Inventory can be viewed in Engine\stimuli.py  **

1. Semantics Rule: - ex. nature, object, people etc.
2. yntactic: Adjective, Noun, or verb
3. Phonological: 1,2, or 3 syllable words
